# 📱 Telecom Plan Recommendation System

A machine learning project that recommends the most suitable telecom plans based on customer calling, messaging, and data usage patterns.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Jupyter Notebook

## 📊 Features
- Data preprocessing and feature engineering
- Customer segmentation using K-Means Clustering
- Telecom plan recommendation using XGBoost
- Model evaluation and performance analysis

## 📂 Dataset
Telecom customer usage dataset (CDR)

## ▶️ Run
```bash
pip install -r requirements.txt
jupyter notebook
```

Open the notebook and run all cells.
